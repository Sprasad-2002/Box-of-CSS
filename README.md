# 🎨 BoxOfCSS

A collection of CSS mini projects and practice tasks built while learning frontend development.
This repository contains multiple small UI components and layouts created using HTML & CSS.

---

## 📂 Project Structure

* `chadcat` – Fun CSS design
* `funnyFace` – Face UI using CSS
* `NavBar` – Navigation bar design
* `Task1` – Basic CSS task
* `Task2` – Practice layout
* `TaskSecond` – Additional layout practice
* `Flex` – Flexbox concepts
* `t2` – CSS practice
* `ishow` – UI component
* `dogDoubt` – Fun experiment
* `gigachad` – Styling practice
* `SAdtiger` – Creative CSS
* `wood` – Texture/UI design

---

## 🚀 Features

* Pure HTML & CSS projects
* Beginner to intermediate level practice
* Covers:

  * Flexbox
  * Layout design
  * Styling techniques
  * UI components

---

## 🛠️ Tech Stack

* HTML5
* CSS3

## 📸 Preview

Each folder contains individual projects.
Open any `.html` file in your browser to view the output.

## 📌 How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/BoxOfCSS.git
   ```

2. Open any project folder

3. Run the HTML file in browser
